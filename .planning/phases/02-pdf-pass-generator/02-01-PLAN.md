<objective>
Rewrite the PDF pass generator in Code.gs to produce a branded, printable, landscape half-page event-ticket PDF for each child in a registration.

Purpose: Phase 2 of 3 — pass generation must be complete before Phase 3 can attach PDFs to the approval email. The current stubs (portrait card, no stub layout, no logo, absent picnic section when consent = "No") do not satisfy the design decisions.

Output: `apps-script/Code.gs` with:
- `LOGO_DRIVE_FILE_ID` top-level constant (placeholder empty string — admin fills it in)

- Rewritten `generatePassPdf(childName, childAge, picnicConsent, submissionId, logo)` with cleanup guard
- Rewritten `generatePassesForRegistration(data)` that fetches logo once and passes it to each child
- Rewritten `buildPassHtml(childName, childAge, picnicConsent, submissionId, logo)` producing a landscape A5 ticket with table layout, jungle green + gold branding, always-present picnic band, and `print-color-adjust: exact` on all colored elements
</objective>

<task type="auto">
  <name>Task 2: Rewrite buildPassHtml() as landscape A5 event ticket with full branding</name>
  <files>apps-script/Code.gs</files>
  <action>
Replace the existing `buildPassHtml(childName, childAge, picnicConsent)` function entirely with a new implementation. New signature: `buildPassHtml(childName, childAge, picnicConsent, submissionId, logo)`.

The function must produce an HTML string with these exact characteristics:

**Page setup (in `<style>` block):**
```css
* { box-sizing: border-box; print-color-adjust: exact; -webkit-print-color-adjust: exact; }
@page { size: A5 landscape; margin: 8mm; }
body { margin: 0; padding: 0; font-family: Arial, Helvetica, sans-serif; background: #ffffff; }
```
This is critical — without `print-color-adjust: exact` the green stub and gold band will render white in the PDF.

**Layout: HTML `<table>` with two `<td>` cells (NOT flexbox or grid):**
- `<table>` with `width:100%; border-collapse:collapse; height:100%;`
- Left `<td>` (stub): width 28%, `background:#1a5c38`, `color:#ffffff`, `vertical-align:middle`, `text-align:center`, `padding:16px 10px`
  - Must include `print-color-adjust:exact; -webkit-print-color-adjust:exact` as inline style (belt-and-suspenders with global rule)
- Right `<td>` (body): width 72%, `background:#ffffff`, `vertical-align:top`, `padding:16px 18px`

**Stub content (left column):**
1. Logo: if `logo.ok` is true, render `<img src="[logo.src]" style="height:40px;margin-bottom:8px;display:block;margin-left:auto;margin-right:auto;" />`. If `logo.ok` is false, render nothing.
2. Decorative jungle element: render an inline SVG leaf silhouette at low opacity as a subtle accent. Use a simple SVG path at `opacity:0.18` positioned before the child name. Do NOT use CSS `::before`/`::after` pseudo-elements (unreliable in WebKit PDF).
3. Child name: `<div style="font-size:22px;font-weight:700;line-height:1.2;margin-bottom:6px;">{childName}</div>`
4. Age: `<div style="font-size:13px;font-weight:600;color:#a3d9b1;">Age: {childAge} yrs</div>`
5. Label: `<div style="margin-top:14px;font-size:9px;letter-spacing:0.15em;text-transform:uppercase;color:#5a9c76;">Admission Pass</div>`

**Body content (right column):**
1. Church name (small caps label): `<div style="font-size:9px;font-weight:700;letter-spacing:0.2em;text-transform:uppercase;color:#1a5c38;margin-bottom:2px;">El Bethel AG International Church</div>`
2. Event title: `<div style="font-size:18px;font-weight:700;color:#c9a227;margin-bottom:10px;">Vacation Bible School 2026</div>`
3. Dates: `<div style="font-size:12px;font-weight:700;color:#1a5c38;margin-bottom:2px;">May 11 - 15, 2026</div>`
4. Time: `<div style="font-size:11px;color:#374151;margin-bottom:8px;">Monday to Friday &nbsp;·&nbsp; 10:00 AM - 1:00 PM</div>`
5. Address: `<div style="font-size:10px;color:#6b7280;margin-bottom:10px;">107, 5th Cross Rd, Chinnapanahalli Main Rd,<br>Doddanekundi, Marathahalli, Bengaluru 560037</div>`
6. Submission ID: `<div style="font-size:10px;color:#9ca3af;">Ref: {submissionId}</div>`

**Picnic band (always rendered, never absent — per CONTEXT.md decision):**

Compute styles before building the HTML string:
```javascript
var picnicActive = (picnicConsent === 'Yes');
var picnicStyle = picnicActive
  ? 'background:#c9a227;color:#1a1a1a;font-weight:700;print-color-adjust:exact;-webkit-print-color-adjust:exact;'
  : 'background:#cccccc;color:#888888;opacity:0.45;text-decoration:line-through;print-color-adjust:exact;-webkit-print-color-adjust:exact;';
```

Render the band at the bottom of the body column, concatenating picnicStyle into the style string:
```
'<div style="margin-top:12px;padding:6px 12px;text-align:center;font-size:11px;'
+ 'letter-spacing:0.1em;text-transform:uppercase;border-radius:4px;' + picnicStyle + '">'
+ 'Day 5 - Full Day Picnic'
+ '</div>'
```

Important: Use string concatenation (the existing Code.gs style) to build the HTML — no template literals — since Apps Script V8 supports template literals but string concatenation is safer and matches the codebase convention.

Do NOT include the vercel URL (`vbs-chi.vercel.app`) anywhere — per STATE.md, Phase 4 is planned to remove it.
